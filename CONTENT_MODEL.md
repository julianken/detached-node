# Content Model (Abstract)

## Core Page Types

### 1) Post
Purpose: A long-form entry with a consistent, repeatable structure.

Fields (abstract):
- title
- slug
- summary
- published_at
- tags
- body
- references (optional)

### 2) Listing
Purpose: A curated list of posts.

Fields (abstract):
- title
- description
- items (post references)

### 3) Static Page
Purpose: Fixed content (e.g., About, Method).

Fields (abstract):
- title
- slug
- body
